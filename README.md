# Alex's Robot
## Personalized, Automated Daily Digest Script

Technology used:
* Python3
* SQL
* Django
* Jinja2
* Gmail API
* PRAW: The Python Reddit API Wrapper
* BeautifulSoup
* Bash scripting
* Various publicly available APIs for updated news, weather and horoscope data
