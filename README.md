# Alex's Robot
## Personalized, automated daily digest of selected web content emailed to a selected group.

## Technology used:
* Python3
* SQL
* Django
* Jinja2
* Gmail API
* PRAW: The Python Reddit API Wrapper
* BeautifulSoup
* Bash scripting
* Various publicly available APIs for updated news, weather and horoscope data
